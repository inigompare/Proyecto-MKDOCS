# Conclusiones

Hemos logrado configurar un servidor SSH seguro sobre Windows sin necesidad de utilizar Linux.

## Puntos Clave

- **Integración:** Windows ahora soporta comandos nativos de SSH.
- **Seguridad:** Hemos aprendido a localizar el archivo `sshd_config` en Windows (`ProgramData`) y a modificar sus reglas.
- **Firewall:** Hemos utilizado PowerShell para gestionar las reglas de red en lugar de la interfaz gráfica.
